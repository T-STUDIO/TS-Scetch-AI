手書きスケッチやイラスト化したいものを撮影して、編集しやすいSVG形式のイラストを手軽に作成できます。

・作画タッチの選択でイラスト化

・写真から既存イラストを撮影し、特定の要素を抽出

・要素毎にレイヤー化

・イラスト化パラメーターによる調整

・パラメーターへのプロンプト追加（音声対応）

・SVG形式での書き出し。

・作画パラメーターの保存機能（同一のタッチでイラスト化）

・BYOK対応（GeminiAPI使用、キーはローカルに安全に保存（サーバ保存なし）、キー取得補助機能）

## 公開アドレス

https://t-studio.github.io/TS-Scetch-AI/

初回のみGeminiAPIキーを登録（もしくは取得補助機能で取得後登録）を行うとローカルに安全に保存されてAIを活用したイラスト作成が可能です。
